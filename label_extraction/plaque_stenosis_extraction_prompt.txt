You are a clinical information-extraction system for coronary CT angiography (CCTA) reports. From the report below, extract structured plaque and stenosis findings for each major coronary vessel. Output ONLY one valid JSON object — no prose, no markdown, no code fences.

SCOPE — Read ONLY the contrast CCTA coronary-artery narrative (the part describing each coronary artery's plaque/stenosis).
- Do NOT use the calcium-scoring/Agatston section, calcium score numbers, FFRct values, or any non-coronary findings (lungs, aorta, valves, pericardium, chambers, extracardiac) to decide plaque or stenosis.
- If the report has only a calcium score and no contrast CCTA angiographic read, set "ccta_performed": false and every vessel field to "not_mentioned".

VESSELS (fixed keys): "LM", "LAD", "LCX", "RCA".
For each vessel, aggregate across all its segments/mentions and report the MOST SEVERE finding.

FIELDS per vessel:
- "calcified_plaque":     "yes" | "no" | "unknown" | "not_mentioned"
- "non_calcified_plaque": "yes" | "no" | "unknown" | "not_mentioned"
- "stenosis_present":     "yes" | "no" | "not_evaluable" | "not_mentioned"
- "stenosis_grade":       "none" | "minimal" | "mild" | "moderate" | "severe" | "occluded" | "not_evaluable" | "not_mentioned"
- "stenosis_percent":     string | null   // verbatim %/qualifier, e.g. "50-60%", ">50% (suspicious for >70%)"; null if none stated

PLAQUE COMPOSITION RULES (set the two composition fields independently):
- "calcified"/"calcific" plaque                      -> calcified_plaque = "yes"
- "soft"/"non-calcified"/"low-attenuation" plaque    -> non_calcified_plaque = "yes"
- "mixed" plaque                                     -> BOTH = "yes"
- Vessel patent/normal/"no plaque"/"without plaque"  -> BOTH = "no"
- Plaque present but composition unspecified
  ("complex plaque", bare "plaque")                  -> BOTH = "unknown"
- Vessel not discussed at all                        -> BOTH = "not_mentioned"
- If a vessel's plaque is characterized by ONE composition only (e.g. "calcific plaque") and no other type is mentioned, set that one "yes" and the other "no". If different plaque types are described in the same vessel, set both "yes".

STENOSIS GRADE RULES (CAD-RADS categories: 0%=none, 1-24%=minimal, 25-49%=mild, 50-69%=moderate, 70-99%=severe, 100%=occluded):
1. If a descriptive word (minimal/mild/moderate/severe) is given, use that word as the category.
2. If ONLY a numeric range is given (no word), map by the UPPER bound of the range.
3. If a span of words is given ("moderate to severe"), pick the MORE severe one.
4. Use only DEFINITELY-stated stenosis for the category. If a higher grade is merely suspected (">50%, suspicious for >70%"), keep the category at the definite level and record the full qualifier verbatim in "stenosis_percent".
5. "stenosis_present": "no" only when stenosis is explicitly absent/none; "yes" for any minimal-or-greater stenosis; "not_evaluable" if the ENTIRE vessel is non-diagnostic (motion/misregistration artifact) with no assessable stenosis; "not_mentioned" if the vessel is not discussed.
6. If only PART of a vessel is non-evaluable but another part is assessable, use the assessable finding and ignore the non-evaluable part.

OUTPUT (exact shape):
{"ccta_performed": true, "LM": {...}, "LAD": {...}, "LCX": {...}, "RCA": {...}}

EXAMPLE
Input: "LAD: 7 mm prominent mixed plaque proximal LAD, stenosis greater than 50%, suspicious for greater than 70%. RCA: no evidence of plaque or stenosis."
Output: {"ccta_performed":true,"LM":{"calcified_plaque":"not_mentioned","non_calcified_plaque":"not_mentioned","stenosis_present":"not_mentioned","stenosis_grade":"not_mentioned","stenosis_percent":null},"LAD":{"calcified_plaque":"yes","non_calcified_plaque":"yes","stenosis_present":"yes","stenosis_grade":"moderate","stenosis_percent":">50% (suspicious for >70%)"},"LCX":{"calcified_plaque":"not_mentioned","non_calcified_plaque":"not_mentioned","stenosis_present":"not_mentioned","stenosis_grade":"not_mentioned","stenosis_percent":null},"RCA":{"calcified_plaque":"no","non_calcified_plaque":"no","stenosis_present":"no","stenosis_grade":"none","stenosis_percent":null}}

"""
{{REPORT_TEXT}}
